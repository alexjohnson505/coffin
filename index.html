About Coffin

Coffin is a UI component built on top of the Skeleton framework, providing a simple, collapsible left shelf. It's fully responsive and automatically collapses in mobile views - allowing you to swipe away and toggle a nagivation in and out of view.

Versioning

Coffin stictly follows semver as defined here http://semver.org/

Dependencies

Coffin was built to work on top of zepto.js, but should work with jquery as well if you're already using that.

Getting Started

Using coffin is super easy. Just define a .coffin, a .stage, and an element to toggle your coffin into view (data-slide="drawer").

Assets

To use coffin first make sure you include skeleton's CSS as well as a link to zepto or jquery. You'll want to have your assets loaded in a similar order as to what is shown directly below.

Markup

There are three main things to note when marking up a page with a coffin.

  1. data-slide="coffin" - this element will toggle the coffin in and out of view when at a mobile size. You can place this attribute on any element, but we do provide a .coffin-tab style as a very simple ui tab which will only be shown in mobile views.
  2. .stage - a wrapper for the container element. Used to controll the sliding of the main content of a page. Note: Containers within a .stage class work off a 12-column grid rather than a 16-column grid, which is different than a standard skeleton implementation and done in order to make space for an open coffin.
  3. .coffin - the coffin is the side shelf. It can contain a .pillow with is a top title bar as well as a ul.nav.
